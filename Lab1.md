# cd
**With No Arguments**
```
{
  [user@sahara ~]$ cd
  [user@sahara ~]$ 
}
```
**With a path to a directory**
```
{
  [user@sahara ~]$ cd lecture1
  [user@sahara ~/lecture1]$ 
}
```
**With a path to a file**
```
{
  [user@sahara ~/lecture1]$ cd messages/en-us.txt
  bash: cd: messages/en-us.txt: Not a directory
}
```
